# LinguaBridge

**多言語データ向けLLMアノテーションプラットフォーム**

> AI企業と熟練アノテーターをつなぐ専門アノテーションプラットフォーム。アフリカ・東南アジア言語に特化。

---

## 概要

LinguaBridgeは以下のユーザー向けに設計されたLLMアノテーションプラットフォームです：
- **AI/MLスタートアップ** - 多言語モデルを構築
- **研究機関** - 学習データを収集
- **アノテーション会社** - 信頼できる外注パートナーを探している

### 主要機能

- **多言語対応**: 英語 + アフリカ言語（アカン語/トウィ、ハウサ語、ヨルバ語、イボ語、ナイジェリアン・ピジン）
- **バッチ処理**: 1,000〜10,000以上のタスクを一括アップロード
- **品質管理**: 評価者間一致度、ゴールドスタンダード検証
- **競争力のある価格**: 主要競合（Scale AI、Appen）より50%安価
- **API + Web UI**: プログラマティックと手動ワークフローの両方をサポート
- **アノテーションタイプ**: テキスト分類、NER、RLHFランキング、翻訳検証

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| **フロントエンド** | React 18 + TypeScript + Tailwind CSS |
| **バックエンド** | Node.js 20 + Express |
| **データベース** | Supabase (PostgreSQL) |
| **認証** | Supabase Auth |
| **決済** | Stripe Connect |
| **ホスティング** | Vercel (フロントエンド) + Railway (バックエンド) |

## はじめに

### 前提条件
- Node.js 20 LTS
- Git
- Supabaseアカウント
- Stripeアカウント

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/your-org/linguabridge.git
cd linguabridge

# フロントエンドセットアップ
cd frontend
npm install
cp .env.example .env
# .envにSupabaseの認証情報を設定
npm run dev

# バックエンドセットアップ（新しいターミナル）
cd backend
npm install
cp .env.example .env
# .envにSupabaseとStripeの認証情報を設定
npm run dev
```

## アノテーションタイプ

| タイプ | 説明 | ユースケース |
|--------|------|-------------|
| **テキスト分類** | テキストをラベルに分類 | 感情分析、トピック分類、安全性 |
| **固有表現認識** | テキスト内のエンティティをタグ付け | NERモデル、情報抽出 |
| **比較ランキング** | 複数の回答をランク付け | RLHF、選好学習 |
| **テキスト評価** | テキスト品質を評価（1-5） | AI出力品質評価 |
| **翻訳検証** | 翻訳品質を検証 | 機械翻訳品質管理 |
