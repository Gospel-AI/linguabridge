# 法的コンプライアンス・対応手順書

**目的**: TaskBridgeプラットフォームの法的枠組みとインシデント対応手順
**バージョン**: 1.0
**最終更新**: 2025年11月1日
**管轄**: 国際（主要: 米国、EU、ガーナ、日本）

---

## 🔴 重要: プラットフォーム責任保護

### セーフハーバー条項

TaskBridgeは以下の法的枠組みに基づく**中立的な仲介プラットフォーム**として運営されます:

**米国**:
- **通信品位法第230条**: ユーザー生成コンテンツに対する責任からの保護
- **デジタルミレニアム著作権法(DMCA)**: 著作権侵害に対するセーフハーバー（適切な通知と削除手続きを実施）

**欧州連合**:
- **デジタルサービス法(DSA)**: 仲介サービスプロバイダーとしての条件付き責任免除
- **電子商取引指令**: ホスティングサービスプロバイダーの免責

**保護を受けるための要件**:
1. ✅ 違法行為の実際の認識がないこと
2. ✅ 認識した際の迅速な削除
3. ✅ 違法取引への積極的関与がないこと
4. ✅ 適切な通報メカニズムの実装

## ⚠️ 禁止行為（利用規約）

### 厳格に禁止されるコンテンツ

以下の行為は**明示的に禁止**され、即座にアカウント停止および法的措置の対象となります:

#### 1. 違法薬物・規制物質
- 違法薬物の製造、流通、販売
- 処方薬の不正取引
- 薬物関連器具の販売

#### 2. 武器・危険物
- 違法武器、銃器、爆発物
- 弾薬販売
- 武器・爆発物の製造方法の指示

#### 3. 詐欺・金融犯罪
- マネーロンダリングスキーム
- 偽造通貨、文書、商品
- クレジットカード詐欺、身元詐称
- ねずみ講、ポンジスキーム
- 脱税支援

#### 4. プライバシー・データ侵害
- ハッキング、不正アクセス
- 個人データ窃盗、ドキシング
- ストーキング、同意なき監視
- 盗難認証情報の販売

#### 5. 児童保護
- 未成年者の搾取に関するあらゆるコンテンツ
- 児童性的虐待資料(CSAM)
- グルーミングや未成年者との不適切な接触

#### 6. 暴力・過激主義
- テロリズムの促進
- 暴力的過激主義コンテンツ
- ヘイトスピーチ、暴力の扇動

#### 7. 知的財産権侵害
- 著作権侵害（海賊版コンテンツ）
- 商標違反
- 偽造品の販売

#### 8. プラットフォームの完全性
- プラットフォーム手数料の回避
- 偽レビュー、評価操作
- スパム、フィッシング攻撃

## 🚨 インシデント対応手順

### レベル1: 重大違反（即座の対応が必要）

**トリガー**:
- 児童保護違反
- テロリズム・暴力コンテンツ
- 進行中の詐欺・詐欺行為
- 法執行機関からの要請

**対応期限**: **2時間以内**

**対応措置**:
```yaml
即座の対応:
  1_コンテンツ削除:
    - タスクを即座に停止し、プラットフォームから非表示
    - 関連するユーザーアカウントを停止
    - 保留中の支払いを凍結

  2_証拠保全:
    - 以下の完全なデータベーススナップショットを取得:
      - タスクコンテンツ（タイトル、説明、添付ファイル）
      - ユーザーアカウント情報（メール、IPアドレス、登録データ）
      - 支払い記録（Stripe取引ID、金額）
      - 当事者間のメッセージ履歴
      - 監査ログ（ユーザーのすべてのアクション）
    - アクセス制限付きで安全に保管
    - 証拠の連鎖管理を文書化

  3_法的通知:
    - 法務顧問に即座に通知
    - インシデントレポートを準備
    - 法律で義務付けられている場合は法執行機関に連絡

  4_関係者への連絡:
    - プラットフォーム管理者に通知
    - 実施した措置をすべて文書化
    - 潜在的な問い合わせへの対応を準備
```

---

### レベル2: 深刻な違反（24時間以内の対応）

**トリガー**:
- 詐欺スキーム
- プライバシー侵害
- 知的財産権侵害
- 継続的なポリシー違反

**対応期限**: **24時間以内**

**対応措置**:
```yaml
深刻な違反への対応:
  1_調査:
    - フラグ付きコンテンツをすべてレビュー
    - 深刻度と範囲を評価
    - 影響を受ける当事者を特定

  2_コンテンツへの対応:
    - レビュー待ちの間、コンテンツを停止
    - コンテンツ作成者に違反を通知
    - 説明または修正を要求

  3_アカウントへの対応:
    - 一時停止（7-30日）または永久禁止
    - 該当する場合、影響を受けた当事者に返金
    - 決定の根拠を文書化

  4_法的レビュー:
    - 法務顧問に相談
    - 法執行機関への通知が必要かどうかを判断
    - プラットフォームの責任リスクを評価
```

---

### レベル3: 軽微な違反（7日以内の対応）

**トリガー**:
- スパム、低品質コンテンツ
- 軽微なポリシー違反
- 不適切だが違法ではないコンテンツ

**対応期限**: **7日以内**

**対応措置**:
```yaml
軽微な違反への対応:
  1_警告システム:
    - ユーザーに正式な警告を発行
    - 違反とポリシーを説明
    - コンプライアンスを要求

  2_コンテンツモデレーション:
    - 違反コンテンツを削除または編集
    - 繰り返し違反がないかユーザーを監視

  3_エスカレーションパス:
    - 3回の警告 → 一時停止
    - 継続的な違反 → 永久禁止
```

## 📋 法執行機関との協力

### 法的要請手順

**有効な要請に必要な内容**:
1. ✅ 公式レターヘッドと権限のある署名
2. ✅ 特定のケース・捜査番号
3. ✅ 求められる情報の明確な記述
4. ✅ 法的根拠（令状、召喚状、裁判所命令）
5. ✅ 管轄区域と適用される法律

### 対応プロセス

```yaml
法執行機関からの要請への対応:
  1_検証:
    - 要請の真正性を確認
    - 管轄権と法的権限を確認
    - 法務顧問に相談

  2_範囲評価:
    - 保有するデータを特定
    - 開示の法的義務を評価
    - プライバシー上の懸念を特定

  3_データ提供:
    - 特に要請されたデータのみを提供
    - 安全な形式で提供
    - 機密性を維持
    - 開示を文書化

  4_ユーザーへの通知:
    - 法律で禁止されていない限り、影響を受けるユーザーに通知
    - 開示の性質を説明
    - 法的要請のコピーを提供（許可されている場合）
```

**法的要請の連絡先**:
- **メール**: legal@taskbridge.com（設定予定）
- **対応時間**: 5-10営業日
- **緊急要請**: 24-48時間以内に迅速対応

## 🌍 国際的コンプライアンス

### 複数管轄区域の考慮事項

**米国**:
- 連邦法の遵守（CFAA、ECPA、CAN-SPAM）
- 州固有の規制（カリフォルニア州CCPA等）
- 金融規制（該当する場合はFinCEN報告）

**欧州連合**:
- GDPR遵守（データ保護、ユーザー権利）
- DSA遵守（コンテンツモデレーション、透明性）
- 電子商取引指令（仲介者責任）

**ガーナ**:
- 電子取引法2008
- データ保護法2012
- サイバーセキュリティ法2020

**日本**:
- 個人情報保護法（APPI）
- 特定電気通信役務提供者の損害賠償責任の制限及び発信者情報の開示に関する法律（プロバイダ責任制限法）
- 資金決済法

### データ転送コンプライアンス

**国境を越えたデータ転送**:
- EUデータには標準契約条項(SCC)を使用
- 十分性認定に準拠
- 転送の法的根拠を文書化

## ⚖️ 法的免責事項（利用規約用）

### 推奨文言

```markdown
## プラットフォーム責任

TaskBridgeは企業とワーカーを結びつける仲介サービスとして運営されています。
当プラットフォームは以下を行いません:

- 企業が投稿するタスクを管理、承認、または推奨すること
- タスクの合法性、安全性、適切性を保証すること
- 支払い処理を超える取引への参加または促進
- ユーザーの行為またはコンテンツに対する責任を負うこと

当プラットフォームはコンテンツを積極的に監視・モデレートしていますが、
すべての禁止コンテンツが検出されることを保証することはできません。
ユーザーはすべての適用法令を遵守する責任を負います。

## 違反の報告

違法または禁止されたコンテンツに遭遇した場合は、当プラットフォームの
報告システムを使用して直ちに報告してください。調査の上、適切な措置を
講じます。

## 法執行機関との協力

当プラットフォームは法執行機関と協力し、ユーザー情報に関する有効な
法的要請に応じます。法律で義務付けられている場合、または危害を防ぐ
ために、ユーザーデータを開示することがあります。

## 補償

ユーザーは、プラットフォームの利用またはポリシー違反から生じる
あらゆる請求、損害、または法的措置から、TaskBridgeを補償し、
免責することに同意します。
```

**この文書は一般的なガイダンスを提供するものであり、法的助言を構成するものではありません。**
