# ものづくり補助金 グローバル枠 申請要件チェックリスト

このチェックリストを使用して、申請前に要件を満たしているか確認してください。

---

## 1. 基本要件（すべて必須）

### 1.1 事業者要件

- [ ] **中小企業・小規模事業者・小規模企業者である**
  - 製造業・建設業・運輸業等：資本金3億円以下または従業員数300人以下
  - 卸売業：資本金1億円以下または従業員数100人以下
  - サービス業：資本金5千万円以下または従業員数100人以下
  - 小売業：資本金5千万円以下または従業員数50人以下
  - ゴム製品製造業：資本金3億円以下または従業員数900人以下
  - ソフトウェア業・情報処理サービス業：資本金3億円以下または従業員数300人以下
  - 旅館業：資本金5千万円以下または従業員数200人以下

- [ ] **日本国内に本社及び補助事業の実施場所を有する**

- [ ] **交付決定時点で、補助事業を実施する法人・個人事業主である**

### 1.2 賃金引上げ要件

- [ ] **給与支給総額を年率平均1.5%以上増加させる計画を有する**
  - 補助事業実施年度の翌年度の3月末時点で達成
  - 基準年度は補助事業実施年度の前年度

- [ ] **事業場内最低賃金を毎年、地域別最低賃金+30円以上の水準にする計画を有する**
  - 補助事業実施年度の翌年度の3月末時点で達成

### 1.3 事業計画要件

- [ ] **補助事業終了後3～5年で付加価値額を年率平均3.0%以上増加させる計画を有する**

- [ ] **補助事業終了後3～5年で給与支給総額を年率平均1.5%以上増加させる計画を有する**

- [ ] **補助事業終了後3～5年で事業場内最低賃金を毎年、地域別最低賃金+30円以上の水準にする計画を有する**

### 1.4 設備投資要件

- [ ] **単価50万円（税抜）以上の機械装置等を取得する**
  - 補助事業実施期間内に納品・検収等を完了すること
  - 適切な管理を行うこと

### 1.5 経費要件

- [ ] **補助対象経費（税抜）が、事業に要する経費（税込）の3分の2以上である**

- [ ] **交付決定日よりも前に発注・契約・購入を行った経費は含まない**

- [ ] **補助事業実施期間内に支払いを完了したことを確認できる**

## 2. グローバル枠固有要件

### 2.1 海外事業の該当性（いずれか1つ以上に該当）

- [ ] **A. 海外への直接投資に関する事業**
  - 海外に子会社・関連会社を設立する
  - 海外企業と資本提携を行う
  - その他、海外への直接投資を行う

- [ ] **B. 海外市場開拓（輸出）に関する事業**
  - 海外への製品・サービスの輸出を行う
  - 海外での販路開拓活動を行う
  - 海外市場向けの製品・サービス開発を行う

- [ ] **C. インバウンド対応に関する事業**
  - 訪日外国人観光客の受入体制整備を行う
  - 外国人向けサービスの提供を行う
  - 多言語対応の環境整備を行う

- [ ] **D. 海外企業との共同で行う事業**
  - 海外企業と共同で製品・サービス開発を行う
  - 海外企業と共同で事業を実施する
  - 海外企業との技術提携を行う

### 2.2 国内生産性向上との関連性

- [ ] **海外事業の実施により、国内の生産性を高める取り組みである**
  - 国内拠点での生産性向上効果が明確に説明できる
  - 海外事業と国内事業の連携が具体的に説明できる

### 2.3 グローバル枠限定経費の使用（該当する場合）

**海外市場開拓（輸出）に関する事業の場合のみ**

- [ ] **海外旅費を使用する場合、補助対象経費総額の1/5以内である**

- [ ] **通訳・翻訳費を使用する場合、補助対象経費総額の1/5以内である**

- [ ] **広告宣伝・販売促進費を使用する場合、補助対象経費総額の1/2以内である**

## 3. 特例措置の適用（該当する場合）

### 3.1 大幅な賃上げに係る補助上限額引上げの特例（該当する場合）

- [ ] **給与支給総額の年平均成長率を6.0%以上増加させる計画を有する**

- [ ] **事業所内最低賃金を毎年、地域別最低賃金+50円以上の水準にする計画を有する**

**引上げ額の確認**：
- 従業員1～5人：+100万円
- 従業員6～20人：+250万円
- 従業員21～50人：+1,000万円
- 従業員51人以上：+1,000万円

### 3.2 最低賃金引上げに係る補助率引上げの特例（該当する場合）

- [ ] **所定の賃金水準を満たしている**
  - 詳細は公募要領p.18を参照

- [ ] **最低賃金の引上げに取り組む計画を有する**

**特例適用時の補助率**：2/3

## 4. 事業計画の適切性

### 4.1 技術面の適切性

- [ ] **補助事業で行う事業内容が明確である**

- [ ] **補助事業の目標が具体的である**

- [ ] **補助事業の実施体制が適切である**

- [ ] **補助事業のスケジュールが実現可能である**

### 4.2 事業化の見通し

- [ ] **市場ニーズが明確である**

- [ ] **顧客・販路が具体的である**

- [ ] **価格の妥当性が説明できる**

- [ ] **収益計画が現実的である**

### 4.3 政策的観点

- [ ] **デジタル技術の活用が含まれる**（加点対象）

- [ ] **グリーン成長戦略への貢献が含まれる**（加点対象）

- [ ] **外部専門家の活用が含まれる**（加点対象）

## 5. 必要書類の準備

### 5.1 基本書類

- [ ] **事業計画書（様式1）**
  - 10ページ以内（図表含む）
  - 補助事業の具体的内容、技術的課題、事業化の見通しを記載

- [ ] **賃金引上げ計画の誓約書（様式2）**
  - 給与支給総額、事業場内最低賃金の計画を記載

- [ ] **決算書等（直近2期分）**
  - 法人：確定申告書別表一、決算報告書（貸借対照表・損益計算書）
  - 個人：確定申告書第一表、所得税青色申告決算書

- [ ] **従業員数を証明する書類**
  - 労働基準法に基づく労働者名簿等

### 5.2 グローバル枠固有書類

- [ ] **海外事業の実施を証明する書類**
  - 海外取引契約書、海外企業との提携契約書等
  - または、海外事業の実施計画書（様式任意）

- [ ] **国内生産性向上との関連性を示す資料**
  - 事業計画書内で明確に説明

### 5.3 加点書類（該当する場合）

- [ ] **賃上げ表明書（加点対象）**

- [ ] **再生事業者であることを証明する書類**

- [ ] **特定事業者であることを証明する書類**

## 6. 申請システム準備

### 6.1 GビズIDプライムアカウント

- [ ] **GビズIDプライムアカウントを取得済み**
  - 取得には2～3週間かかるため、早めに申請
  - https://gbiz-id.go.jp/top/

### 6.2 電子申請システム

- [ ] **Jグランツアカウントを準備済み**
  - GビズIDと連携して使用
  - https://www.jgrants-portal.go.jp/

## 7. スケジュール確認

- [ ] **公募期間内に申請可能である**
  - 公募期間：2025年10月24日（金）～2026年1月30日（金）17:00
  - 電子申請受付開始：2025年12月26日（金）17:00～

- [ ] **補助事業実施期間を理解している**
  - 交付決定日から12か月以内
  - ただし、採択発表日から14か月後の日まで

- [ ] **採択公表時期を把握している**
  - 2026年4月下旬頃予定

## チェック結果

### すべての必須項目にチェックが入りましたか？

- **はい** → 申請準備を進めてください
- **いいえ** → チェックが入らない項目について、要件を満たすための対策を検討してください

### 不明点がある場合

- **事務局への問い合わせ**：公募要領記載の連絡先に問い合わせ
- **認定支援機関への相談**：事業計画書作成支援を受けることも可能
