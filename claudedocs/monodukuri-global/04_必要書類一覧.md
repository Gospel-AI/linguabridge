# ものづくり補助金 グローバル枠 必要書類一覧

申請に必要な書類を漏れなく準備するため、このチェックリストを活用してください。

---

## 📋 必須書類（全申請者共通）

### 1. 事業計画書（様式1）

| 項目 | 詳細 |
|------|------|
| **様式** | 様式1（指定フォーマット） |
| **ページ数** | 10ページ以内（図表含む） |
| **ファイル形式** | PDF |
| **ファイルサイズ** | 20MB以内 |
| **準備のポイント** | [事業計画書作成ガイド](03_事業計画書作成ガイド.md)を参照 |

**記載内容**：
- 補助事業の具体的内容
- 技術的課題とその解決方法
- 事業化の見通し
- 政策的観点

**チェック項目**：
- [ ] 10ページ以内に収まっている
- [ ] 海外事業の内容が具体的に記載されている
- [ ] 国内生産性向上との関連性が明記されている
- [ ] 導入設備の仕様が明記されている（単価50万円以上）
- [ ] 5年間の収益計画が記載されている
- [ ] 付加価値額・給与総額の増加計画が要件を満たしている

---

### 2. 賃金引上げ計画の誓約書（様式2）

| 項目 | 詳細 |
|------|------|
| **様式** | 様式2（指定フォーマット） |
| **ファイル形式** | PDF |
| **作成のポイント** | 要件を満たす数値を正確に記載 |

**記載内容**：
- 給与支給総額の増加計画（年率平均1.5%以上）
- 事業場内最低賃金の計画（地域別最低賃金+30円以上）

**チェック項目**：
- [ ] 基準年度の給与支給総額が正確に記載されている
- [ ] 5年間の増加計画が年率平均1.5%以上になっている
- [ ] 事業場内最低賃金が地域別最低賃金+30円以上になっている
- [ ] 代表者の署名・押印がある

---

### 3. 決算書等（直近2期分）

#### 法人の場合

**必要書類**：
- [ ] **確定申告書別表一**（直近2期分）
  - 税務署の受付印があるもの
  - または、電子申請の受信通知

- [ ] **決算報告書**（直近2期分）
  - 貸借対照表
  - 損益計算書
  - 製造原価報告書（製造業の場合）

**チェック項目**：
- [ ] 直近2期分が揃っている
- [ ] 税務署の受付印または電子申請の受信通知がある
- [ ] 全ページが揃っている（途中ページの欠落がない）

#### 個人事業主の場合

**必要書類**：
- [ ] **確定申告書第一表**（直近2期分）
  - 税務署の受付印があるもの
  - または、電子申請の受信通知

- [ ] **所得税青色申告決算書**または**収支内訳書**（直近2期分）
  - 損益計算書
  - 貸借対照表（青色申告の場合）

**チェック項目**：
- [ ] 直近2期分が揃っている
- [ ] 税務署の受付印または電子申請の受信通知がある
- [ ] 全ページが揃っている

---

### 4. 従業員数を証明する書類

**必要書類（いずれか）**：
- [ ] **労働基準法に基づく労働者名簿**
- [ ] **役員名簿**（役員の人数確認用）

**チェック項目**：
- [ ] 申請時点の従業員数が正確に把握できる
- [ ] 氏名、生年月日、入社年月日などが記載されている
- [ ] 個人情報保護に配慮している（マスキング可）

---

### 5. 海外事業を証明する書類

**必要書類（いずれか、複数可）**：
- [ ] **海外取引契約書**
  - 輸出契約、販売契約など

- [ ] **海外企業との提携契約書**
  - 資本提携、技術提携、共同事業契約など

- [ ] **海外子会社・関連会社の登記簿謄本**（直接投資の場合）

- [ ] **海外事業の実施計画書**（契約書等がない場合）
  - 対象国・地域
  - 事業内容
  - 実施スケジュール
  - 想定される取引先

**チェック項目**：
- [ ] 海外事業の実施が具体的に証明できる
- [ ] 対象国・地域が明記されている
- [ ] 事業内容が明確である

## 📋 加点対象書類（該当する場合）

### 6. 賃上げ表明書（加点対象）

| 項目 | 詳細 |
|------|------|
| **様式** | 任意様式 |
| **ファイル形式** | PDF |
| **加点内容** | 賃上げへの積極的な取り組みを評価 |

**記載内容**：

**チェック項目**：
- [ ] 代表者の署名・押印がある
- [ ] 具体的な数値が記載されている

---

**必要書類**：
- [ ] **中小企業活性化協議会の支援決定通知書**
- [ ] **民事再生計画認可決定書**
- [ ] その他、再生事業者であることを証明する書類

**必要書類**：
- [ ] **事業継続力強化計画認定書**
- [ ] **連携事業継続力強化計画認定書**

## 📋 特例措置適用時の追加書類

### 9. 大幅な賃上げに係る補助上限額引上げの特例

**該当する場合**：
- 給与支給総額を年率平均6.0%以上増加させる
- 事業場内最低賃金を地域別最低賃金+50円以上にする

**必要書類**：
- [ ] **賃金引上げ計画書**（様式2に詳細を記載）
- [ ] **従業員数を証明する書類**（人数規模の確認）

**チェック項目**：
- [ ] 6.0%以上の増加計画になっている
- [ ] 地域別最低賃金+50円以上の計画になっている
- [ ] 従業員数規模に応じた引上げ額を確認している

---

### 10. 最低賃金引上げに係る補助率引上げの特例

**該当する場合**：
- 所定の賃金水準を満たしている事業者が最低賃金引上げに取り組む

**必要書類**：
- [ ] **賃金台帳**（直近3か月分）
- [ ] **最低賃金引上げ計画書**

**チェック項目**：
- [ ] 現在の賃金水準が確認できる
- [ ] 引上げ計画が具体的に記載されている

## 📋 申請システム関連

### 13. GビズIDプライムアカウント

**取得方法**：
1. https://gbiz-id.go.jp/ にアクセス
2. 「GビズIDプライム」を選択
3. 必要事項を入力して申請
4. 審査完了後、ID・パスワードが郵送される（2～3週間）

**チェック項目**：
- [ ] GビズIDプライムアカウントを取得済み
- [ ] ID・パスワードを確認済み
- [ ] ログインできることを確認済み

**重要**：取得には2～3週間かかるため、早めに申請してください。

---

### 14. 電子申請システム（Jグランツ）

**アクセス方法**：
1. https://www.jgrants-portal.go.jp/ にアクセス
2. GビズIDでログイン
3. 「ものづくり・商業・サービス生産性向上促進補助金」を選択

**チェック項目**：
- [ ] Jグランツにログインできる
- [ ] 申請画面にアクセスできる
- [ ] PDFファイルのアップロードができる

## 📝 書類作成の注意事項

### ファイル形式
- すべての書類をPDF形式で準備
- ファイルサイズは各20MB以内
- ファイル名は分かりやすく（例：事業計画書_会社名.pdf）

### 記載内容
- 誤字脱字がないか確認
- 数値に誤りがないか確認
- 税務署の受付印または電子申請の受信通知があるか確認

### 個人情報保護
- 必要に応じてマスキング可能
- ただし、審査に必要な情報は残すこと

### 署名・押印
- 代表者の署名・押印が必要な書類を確認
- 電子署名も可能

---

## 💡 書類準備のコツ

### 1. 早めに準備を始める
- 決算書等は既に準備されているため比較的容易
- 事業計画書の作成には時間がかかる（1～2週間程度）
- GビズID取得には2～3週間かかる

### 2. 認定支援機関に相談
- 事業計画書の作成支援を受けることができる
- 必要書類の確認も依頼できる

### 3. チェックリストを活用
- このチェックリストを印刷して使用
- 準備完了した書類にチェックを入れる
- 申請直前に最終確認
