# ものづくり補助金 グローバル枠 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| **補助金名称** | ものづくり・商業・サービス生産性向上促進補助金（グローバル枠） |
| **公募回** | 第22次公募 |
| **公募期間** | 2025年10月24日（金）～2026年1月30日（金）17:00 |
| **電子申請受付開始** | 2025年12月26日（金）17:00～ |
| **採択公表** | 2026年4月下旬頃予定 |

## グローバル枠の特徴

### 概要
海外事業を実施し、国内の生産性を高める取り組みに必要な設備・システム投資等を支援

### 海外事業の定義
以下のいずれかに該当する事業：
1. **海外への直接投資**に関する事業
2. **海外市場開拓（輸出）**に関する事業
3. **インバウンド対応**に関する事業
4. **海外企業との共同**で行う事業

## 補助金額・補助率

| 項目 | 内容 |
|------|------|
| **補助上限額** | 3,000万円 |
| **補助下限額** | 100万円 |
| **補助率** | 中小企業：1/2<br>小規模企業・小規模事業者：2/3 |
| **補助事業実施期間** | 交付決定日から12か月<br>（ただし採択発表日から14か月後の日まで） |

## 補助対象経費

### 必須経費
- **機械装置・システム構築費**（単価50万円以上の設備投資が必須）

### その他の補助対象経費
- 技術導入費
- 専門家経費
- 運搬費
- クラウドサービス利用費
- 原材料費
- 外注費
- 知的財産権等関連経費

### グローバル枠限定の補助対象経費
**海外市場開拓（輸出）に関する事業のみ**
- **海外旅費**（上限：補助対象経費総額の1/5）
- **通訳・翻訳費**（上限：補助対象経費総額の1/5）
- **広告宣伝・販売促進費**（上限：補助対象経費総額の1/2）

### 経費の上限
機械装置・システム構築費以外の経費は、総額で**1,000万円（税抜）まで**

## 重要な注意事項

### 設備投資の必須要件
- **単価50万円（税抜）以上**の機械装置等の取得が必須
- 設備は補助事業実施期間内に納品・検収等を完了すること
- 適切な管理を行うこと

### 経費の条件
- 補助対象経費（税抜）は、事業に要する経費（税込）の**3分の2以上**であること
- 交付決定日よりも前に発注・契約・購入を行った経費は**補助対象外**
- 補助事業実施期間内に支払いを完了したことを確認できるもののみ

### 特例措置

#### C) 大幅な賃上げに係る補助上限額引上げの特例
従業員数規模に応じて補助上限額を引上げ

| 従業員数 | 補助上限引上げ額 |
|---------|----------------|
| 1～5人 | 各枠の上限額から最大 100万円 |
| 6～20人 | 各枠の上限額から最大 250万円 |
| 21～50人 | 各枠の上限額から最大 1,000万円 |
| 51人以上 | 各枠の上限額から最大 1,000万円 |

**条件**：
- 給与支給総額の年平均成長率を6.0%以上増加
- 事業所内最低賃金を毎年、地域別最低賃金+50円以上

#### D) 最低賃金引上げに係る補助率引上げの特例
所定の賃金水準の事業者が最低賃金の引上げに取り組む場合、補助率を**2/3**に引上げ
