# ものづくり補助金 グローバル枠 申請手続き

このガイドでは、申請開始から採択までの手順を詳しく解説します。

---

## 📅 重要な日程

| 項目 | 日程 |
|------|------|
| **公募開始** | 2025年10月24日（金） |
| **公募締切** | 2026年1月30日（金）17:00 |
| **電子申請受付開始** | 2025年12月26日（金）17:00～ |
| **採択公表** | 2026年4月下旬頃予定 |
| **交付決定** | 採択公表後、順次 |
| **補助事業実施期間** | 交付決定日から12か月以内（ただし採択発表日から14か月後の日まで） |

## 📋 Phase 1: 準備期間（2025年10月24日～12月25日）

### ステップ1: 要件確認（所要時間: 1～2時間）

**やること**：
- [ ] [申請要件チェックリスト](02_申請要件チェックリスト.md)で要件を確認
- [ ] 自社が該当するか判断
- [ ] 不明点があれば事務局に問い合わせ

**重要ポイント**：
- すべての基本要件を満たすことが必須
- グローバル枠固有要件（海外事業）の該当性を確認
- 特例措置の適用可否を検討

---

### ステップ2: GビズIDプライム取得（所要時間: 申請15分、取得まで2～3週間）

**重要**：取得には2～3週間かかるため、最優先で対応してください。

**手順**：
1. **GビズID公式サイトにアクセス**
   - https://gbiz-id.go.jp/

2. **「GビズIDプライム」を選択**
   - 「アカウント作成」をクリック
   - 「GビズIDプライム」を選択（※「エントリー」は不可）

3. **必要事項を入力**
   - 法人名
   - 代表者氏名
   - メールアドレス
   - 電話番号
   - 所在地

4. **必要書類をアップロード**
   - 印鑑証明書（発行から3か月以内）
   - 代表者の本人確認書類（運転免許証等）

5. **審査完了を待つ**
   - 申請から2～3週間でID・パスワードが郵送される
   - 到着後、ログインして動作確認

**チェック項目**：
- [ ] GビズIDプライムを取得済み（※エントリーは不可）
- [ ] ID・パスワードを受け取り済み
- [ ] ログインできることを確認済み

---

### ステップ3: 事業計画書作成（所要時間: 1～2週間）

**手順**：
1. **[事業計画書作成ガイド](03_事業計画書作成ガイド.md)を読む**
   - 審査のポイントを理解
   - 推奨される構成を確認

2. **情報収集**
   - 市場データ
   - 競合情報
   - 導入設備の仕様
   - 顧客ニーズ

3. **初稿作成**
   - 推奨構成に沿って作成
   - 図表を効果的に活用
   - 数値の根拠を明確に

4. **レビュー・改善**
   - 社内でレビュー
   - 認定支援機関に相談（推奨）
   - 第三者の意見を取り入れる

5. **最終版作成**
   - 10ページ以内に収める
   - 誤字脱字を確認
   - PDF化して保存

**重要ポイント**：
- 海外事業の内容を具体的に記載
- 国内生産性向上との関連性を明確に
- 要件（付加価値額3.0%増、給与総額1.5%増）を満たす計画にする

**チェック項目**：
- [ ] 事業計画書（様式1）を作成完了
- [ ] 10ページ以内に収まっている
- [ ] PDF形式で保存済み（20MB以内）
- [ ] 認定支援機関のレビューを受けた（推奨）

---

### ステップ4: 必要書類準備（所要時間: 2～3日）

**手順**：
1. **[必要書類一覧](04_必要書類一覧.md)を確認**
   - 必須書類をリストアップ
   - 加点対象書類の該当性を確認

2. **書類を収集**
   - 決算書等（既存書類）
   - 海外事業を証明する書類
   - 従業員数を証明する書類

3. **新規作成書類を準備**
   - 賃金引上げ計画の誓約書（様式2）
   - その他の証明書類

4. **すべてPDF化**
   - スキャンまたはPDF変換
   - ファイルサイズを確認（各20MB以内）
   - ファイル名を分かりやすく設定

**チェック項目**：
- [ ] すべての必須書類を準備完了
- [ ] 加点対象書類を準備完了（該当する場合）
- [ ] すべてPDF形式に変換済み
- [ ] ファイルサイズが適切（各20MB以内）

## 📋 Phase 2: 申請期間（2025年12月26日～2026年1月30日）

### ステップ5: 電子申請システムで申請（所要時間: 1～2時間）

**手順**：

#### 5-1. Jグランツにログイン
1. **Jグランツ公式サイトにアクセス**
   - https://www.jgrants-portal.go.jp/

2. **GビズIDでログイン**
   - GビズIDプライムのID・パスワードを入力

3. **「ものづくり・商業・サービス生産性向上促進補助金」を選択**
   - 補助金一覧から該当する補助金を選択
   - 「グローバル枠」を選択

#### 5-2. 申請情報の入力
1. **基本情報入力**
   - 事業者名
   - 所在地
   - 代表者氏名
   - 連絡先

2. **事業計画の概要入力**
   - 補助事業の概要（簡潔に）
   - 補助金額
   - 補助率

3. **要件の確認**
   - チェックボックスで要件を確認
   - すべての要件を満たしていることを確認

#### 5-3. 書類のアップロード
1. **必須書類をアップロード**
   - 事業計画書（様式1）
   - 賃金引上げ計画の誓約書（様式2）
   - 決算書等
   - 従業員数を証明する書類
   - 海外事業を証明する書類

2. **加点対象書類をアップロード**（該当する場合）
   - 賃上げ表明書
   - その他の証明書類

3. **アップロード完了を確認**
   - ファイルが正常にアップロードされたか確認
   - ファイル名が正しいか確認

#### 5-4. 申請内容の確認
1. **入力内容を確認**
   - 基本情報に誤りがないか
   - 書類が正しくアップロードされているか

2. **確認画面で最終チェック**
   - すべての項目が入力されているか
   - 書類の抜け漏れがないか

#### 5-5. 申請の送信
1. **「申請する」ボタンをクリック**
   - 申請内容を送信

2. **受付番号を控える**
   - 受付番号が表示される
   - スクリーンショットを保存
   - 受付完了メールを確認

**重要ポイント**：
- 締切は2026年1月30日（金）17:00
- 締切間際は混雑するため、余裕を持って申請
- 一度送信すると修正不可（締切前に再申請は可能）

**チェック項目**：
- [ ] Jグランツにログインできた
- [ ] すべての必須書類をアップロードできた
- [ ] 申請を送信できた
- [ ] 受付番号を控えた
- [ ] 受付完了メールを受信した

---

### ステップ6: 申請内容の最終確認（所要時間: 30分）

**手順**：
1. **受付完了メールを確認**
   - 受付番号
   - 申請日時
   - 申請内容の概要

2. **申請内容をダウンロード**（可能な場合）
   - Jグランツから申請内容をダウンロード
   - 印刷またはPDF保存

3. **社内で情報共有**
   - 申請したことを関係者に共有
   - 受付番号を共有

**チェック項目**：
- [ ] 受付完了メールを確認した
- [ ] 受付番号を記録した
- [ ] 申請内容を保存した
- [ ] 関係者に共有した

---

## 📋 Phase 3: 審査期間（2026年2月～4月）

### ステップ7: 事務局による審査（所要期間: 約3か月）

**審査内容**：
- 要件の適合性確認
- 事業計画の妥当性評価
- 政策的観点の評価

**審査の3つの観点**：
1. **技術面**（30点）
   - 補助事業の内容の明確性
   - 技術的課題の適切性
   - 実施体制・スケジュールの妥当性

2. **事業化面**（30点）
   - 市場ニーズの明確性
   - 顧客・販路の具体性
   - 収益計画の現実性

3. **政策面**（40点）
   - 地域経済への貢献
   - デジタル技術の活用
   - グリーン成長戦略への貢献
   - 賃上げへの取り組み
   - **グローバル展開の具体性**

**この期間にやること**：
- 事務局からの連絡に対応できるようにする
- 追加資料の提出依頼があれば速やかに対応

**チェック項目**：
- [ ] 事務局からの連絡を確認できる体制を整えた
- [ ] 追加資料提出に備えて関連書類を整理した

---

### ステップ8: 追加資料提出（必要に応じて）

**事務局から追加資料の提出依頼があった場合**：
1. **依頼内容を確認**
   - どの書類が必要か
   - 提出期限はいつか

2. **速やかに準備**
   - 依頼された書類を準備
   - PDF化

3. **指定された方法で提出**
   - Jグランツまたはメール
   - 提出期限を厳守

**重要ポイント**：
- 追加資料提出依頼に速やかに対応することが重要
- 提出期限を過ぎると審査対象外となる可能性がある

---

## 📋 Phase 4: 採択・交付決定（2026年4月下旬～）

### ステップ9: 採択結果公表（2026年4月下旬頃）

**公表方法**：
- 事務局ホームページで公表
- 採択事業者には個別に通知

**採択された場合**：
1. **採択通知を確認**
   - 受付番号
   - 事業者名
   - 補助金額

2. **社内で情報共有**
   - 採択されたことを関係者に共有
   - 補助事業の実施準備を開始

**不採択だった場合**：
- 次回公募での再申請を検討
- 事業計画の改善点を検討
- 認定支援機関に相談

**チェック項目**：
- [ ] 採択結果を確認した
- [ ] 採択通知を受け取った（採択の場合）
- [ ] 関係者に共有した

---

### ステップ10: 交付決定通知（採択後、順次）

**手順**：
1. **交付申請書の提出**
   - 事務局から交付申請の案内が届く
   - 交付申請書を提出

2. **交付決定通知を受け取る**
   - 事務局から交付決定通知が届く
   - 交付決定日を確認

3. **補助事業の開始**
   - 交付決定日以降に発注・契約を開始
   - **交付決定日より前の発注・契約は補助対象外**

**重要ポイント**：
- 交付決定日より前に発注・契約を行わないこと
- 交付決定日から補助事業実施期間がカウントされる

**チェック項目**：
- [ ] 交付申請書を提出した
- [ ] 交付決定通知を受け取った
- [ ] 交付決定日を確認した
- [ ] 補助事業の実施計画を確認した

---

## 📋 Phase 5: 補助事業実施（交付決定日から12か月以内）

### ステップ11: 補助事業の実施

**実施期間**：
- 交付決定日から12か月以内
- ただし、採択発表日から14か月後の日まで

**やること**：
1. **設備の発注・契約**
   - 交付決定日以降に発注
   - 契約書を保管

2. **設備の納品・検収**
   - 納品を確認
   - 検収書を保管

3. **支払いの完了**
   - 補助事業実施期間内に支払いを完了
   - 領収書・振込明細を保管

4. **進捗報告**（必要に応じて）
   - 事務局から進捗報告を求められた場合に対応

**重要ポイント**：
- すべての経費は補助事業実施期間内に支払いを完了すること
- 契約書、納品書、領収書等の証拠書類を必ず保管

**チェック項目**：
- [ ] 交付決定日以降に発注・契約した
- [ ] 設備の納品・検収を完了した
- [ ] 支払いを完了した
- [ ] すべての証拠書類を保管した

---

### ステップ12: 実績報告（補助事業完了後30日以内）

**手順**：
1. **実績報告書の作成**
   - 補助事業の実施内容
   - 補助対象経費の詳細
   - 成果の概要

2. **証拠書類の添付**
   - 契約書
   - 納品書
   - 領収書
   - 振込明細
   - 写真（設備の設置状況）

3. **実績報告書の提出**
   - 補助事業完了後30日以内に提出
   - Jグランツまたは事務局指定の方法で提出

4. **確定検査**
   - 事務局による確定検査
   - 必要に応じて現地調査

**重要ポイント**：
- 実績報告書の提出期限を厳守
- 証拠書類は漏れなく添付
- 補助対象経費の計算を正確に

**チェック項目**：
- [ ] 実績報告書を作成した
- [ ] 証拠書類をすべて添付した
- [ ] 期限内に提出した
- [ ] 確定検査に対応した

---

### ステップ13: 補助金の交付（確定検査後）

**手順**：
1. **補助金額の確定**
   - 確定検査の結果、補助金額が確定
   - 確定通知を受け取る

2. **補助金の請求**
   - 請求書を提出
   - 振込先口座を指定

3. **補助金の受け取り**
   - 指定口座に振り込まれる
   - 入金を確認

4. **事業化状況報告**
   - 補助事業終了後5年間、毎年報告
   - 売上高、付加価値額、給与総額等を報告

**重要ポイント**：
- 補助金受け取り後も5年間の報告義務がある
- 要件を満たさない場合、補助金の返還を求められることがある

**チェック項目**：
- [ ] 補助金額の確定通知を受け取った
- [ ] 請求書を提出した
- [ ] 補助金の入金を確認した
- [ ] 事業化状況報告の義務を理解した

---

## ⚠️ 注意事項

### 1. 交付決定日より前の発注・契約は補助対象外
- 交付決定日を必ず確認
- それより前の発注・契約は補助対象にならない

### 2. 補助事業実施期間内に支払いを完了
- 期間内に納品・検収・支払いをすべて完了
- 期間を過ぎた経費は補助対象外

### 3. 証拠書類の保管
- 契約書、納品書、領収書等を必ず保管
- 補助事業終了後5年間の保管義務

### 4. 補助金の返還リスク
- 要件を満たさない場合、補助金の返還を求められる
- 虚偽の申請は刑事罰の対象

### 5. 事業化状況の報告義務
- 補助事業終了後5年間、毎年報告
- 報告を怠ると次回申請が制限される
