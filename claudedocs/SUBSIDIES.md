## 概要

TaskBridgeの海外展開を支援するため、日本政府の補助金制度を調査しました。主に2つの補助金が候補となります。

### 調査対象の補助金

| 補助金名 | 主な用途 | 補助額 | TaskBridge適合性 |
|---------|---------|--------|------------------|
| **ものづくり補助金グローバル枠** | 海外市場開拓・システム構築 | 100万～3,000万円 | ⭐⭐⭐⭐☆（4/5） |
| **小規模事業者持続化補助金** | 販路開拓・マーケティング | 50万～250万円 | ⭐⭐⭐⭐⭐（5/5）**推奨** |

### 重要な前提条件

**両補助金に共通**:
- ❌ **原則後払い**：先に自己資金で支払いが必要
- ⏰ **入金まで1年以上**：申請から補助金入金まで1年以上かかる
- 📝 **厳密な書類管理**：8種類の会計証拠書類が必須
- 💰 **資金立て替えが必要**：運転資金の確保が重要

## ものづくり補助金グローバル枠

### 基本情報

| 項目 | 詳細 |
|------|------|
| **正式名称** | ものづくり・商業・サービス生産性向上促進補助金（グローバル市場開拓枠） |
| **目的** | 海外展開やインバウンド対応など、国外の需要を取り込みながら日本国内の生産性向上に資する取組を支援 |
| **補助額** | **100万円～3,000万円** |
| **補助率** | **1/2**（中小企業）、**2/3**（小規模事業者） |
| **下限額** | **100万円**（実質支出200万円以上） |
| **補助事業期間** | 採択公表日から最大12ヶ月 |

### 対象となる事業カテゴリー

以下のいずれかに該当する取組：

1. **海外への直接投資**
   - 海外での製造拠点設立など

2. **海外市場開拓（輸出）** ⭐ **TaskBridgeはこちら**
   - 製品等の最終販売先の2分の1以上が海外顧客
   - 計画期間中の補助事業の売上累計額が補助額を上回る事業計画

3. **インバウンド需要対応**
   - 訪日外国人向けサービス

4. **海外企業との共同事業**
   - 海外企業とのアライアンス

### 申請要件

#### 基本要件（全枠共通）
- **付加価値額増加**：事業終了後3～5年で年平均3.0%以上の成長
- **賃金増加**：給与支給総額の年平均2.0%以上増加
- **最低賃金引き上げ**：毎年地域別最低賃金+30円以上
- **行動計画公表**：従業員21名以上の企業が対象

#### グローバル枠追加要件
1. **実現可能性調査の実施**
   - 市場調査、現地規制調査、信用調査等

2. **専門人材の確保**
   - 社内専門家または外部専門家との連携

3. **金融機関確認書**
   - 資金調達予定の場合

#### 海外市場開拓（輸出）カテゴリー追加要件
- **海外市場調査報告書**（具体的な想定顧客を明記）
- **試作品等の性能評価報告書**

### 補助対象経費

| 経費項目 | TaskBridgeでの該当 | 認められる可能性 | 備考 |
|---------|-------------------|----------------|------|
| **機械装置・システム構築費**（必須） | プラットフォーム開発費用 | ⚠️ **要確認** | **最重要確認事項** |
| 海外旅費 | 北米・欧州への営業出張、展示会 | ✅ 高い | - |
| 広告宣伝・販売促進費 | 海外向けマーケティング | ✅ 高い | 海外市場開拓のみ |
| 通訳・翻訳費 | 多言語対応、契約書翻訳 | ✅ 高い | - |
| 専門家経費 | 法務、税務、マーケティングコンサル | ✅ 高い | - |
| 技術導入費 | 外部技術・ノウハウの導入 | ✅ 高い | - |
| 外注費 | システム開発の外注 | ✅ 高い | - |
| 知的財産権等関連経費 | 特許・商標の取得 | ✅ 高い | - |

### 支払い方式

#### 原則：後払い（精算払）
```
Step 1: 自己資金で全額支払い（先出し）
  ↓
Step 2: 事業完了後、実績報告書提出
  ↓
Step 3: 確定検査（書類審査）
  ↓
Step 4: 補助金振込（約1ヶ月後）

⏰ 申請から入金まで：1年以上
```

#### 例外：概算払い（条件付き）

**利用条件（厳しい）**:
- ✅ すでに**支払い・納品が完了している経費のみ**
- ✅ 事業計画の実施期間内の計画内容が完了している
- ✅ **概算払いを利用しなければ事業が開始できない**などの理由が必要
- ⚠️ **1度しか申請できない**
- ⚠️ **審査が厳しい**

**概算払いの金額**:
```
上限：補助金交付決定額の90%
計算式：支払済み補助対象経費 × 補助率

例：700万円の経費を支払い済み、補助率1/2の場合
→ 概算払い：350万円 × 90% = 315万円
```

### 必須書類（精算時）

8種類の会計証拠書類が必須：

| No. | 書類 | 備考 |
|-----|------|------|
| ① | 見積依頼書（仕様書） | - |
| ② | 見積書 | - |
| ③ | 相見積書 | **税抜単価50万円以上の場合のみ** |
| ④ | 注文書（契約書） | - |
| ⑤ | 受注書 | - |
| ⑥ | 納品書 | - |
| ⑦ | 請求書 | - |
| ⑧ | 振込依頼書（振込金受取書） | **銀行振込の証拠** |

**想定予算例（12ヶ月）**:
```
システム構築費（プラットフォーム開発）：400万円
海外マーケティング・広告費：150万円
海外出張費（展示会、顧客訪問）：100万円
専門家経費（法務、税務、コンサル）：50万円
-------------------------------------------
合計：700万円

実質負担（2/3の場合）：234万円
```

### メリット・デメリット

**メリット**:
- ✅ **補助額が大きい**（最大3,000万円）
- ✅ **システム開発費に制限なし**
- ✅ **海外展開に特化**した支援
- ✅ **専門家経費も対象**

**デメリット**:
- ❌ **下限100万円の壁**（実質200万円以上の支出）
- ❌ **プラットフォーム適格性が不明**（要確認）
- ❌ **申請が複雑**（市場調査報告書など）
- ❌ **大きな資金立て替えが必要**

### 最重要確認事項

**プラットフォーム/SaaSの適格性確認が必須**:

```
❓ 質問1：「システム構築費」にプラットフォーム開発費が含まれるか？
❓ 質問2：SaaSビジネスモデルが「製品等」に該当するか？
❓ 質問3：個人事業主でも応募可能か？（法人設立が必要か？）

→ 中小企業庁または認定支援機関への確認が必須
```

## 小規模事業者持続化補助金

### 基本情報

| 項目 | 詳細 |
|------|------|
| **正式名称** | 小規模事業者持続化補助金（一般型） |
| **目的** | 小規模事業者等が経営計画を自ら策定し、商工会・商工会議所の支援を受けながら取り組む販路開拓等を支援 |
| **補助額** | **50万円**（基本）～**250万円**（特例） |
| **補助率** | **2/3**（基本）、**3/4**（賃上げ特例・赤字事業者） |
| **下限額** | **なし**（使いやすい） |
| **補助事業期間** | 交付決定日から原則10ヶ月 |

### 2025年度の支援類型

4つの類型に整理：

1. **一般型**（通常枠・災害支援枠）⭐ **TaskBridgeはこちら**
2. **創業型**
3. **共同・協業型**
4. **ビジネスコミュニティ型**

### 補助対象経費

| 経費項目 | TaskBridgeでの該当 | 上限・制限 |
|---------|-------------------|-----------|
| **機械装置等費** | - | - |
| **広報費** | 海外向けマーケティング・広告 | - |
| **ウェブサイト関連費** | プラットフォーム改善・開発 | ⚠️ **補助対象経費の1/4まで（最大50万円）** |
| **展示会等出展費** | 海外展示会・商談会 | オンライン含む |
| **旅費** | 海外出張 | - |
| **開発費** | システム開発関連 | ウェブサイト関連費として計上 |
| **借料** | 会場・機器レンタル | - |
| **委託・外注費** | 専門家コンサル | 税理士等への相談含む |

**2025年度の変更点**:
- ❌ **資料購入費**が対象外に
- ❌ **設備処分費**が対象外に

### ウェブサイト関連費の重要な制限

**最大の制約**:
```
制限：補助対象経費の1/4まで
上限：50万円（絶対上限）

例1：補助額50万円の場合
→ ウェブサイト関連費：最大12.5万円
→ 実際の支出：最大18.75万円（補助率2/3）

例2：補助額200万円（特例）の場合
→ ウェブサイト関連費：最大50万円
→ 実際の支出：最大75万円（補助率2/3）
```

**その他の制限**:
- ❌ **単独申請不可**：他の経費項目と組み合わせが必要
- ✅ **OK**：販路開拓、業務効率化が目的
- ❌ **NG**：システム開発自体が目的、内部管理用のシステム

**TaskBridgeでの対応**:
```
✅ 「海外企業向けの販路開拓プラットフォーム」として説明
→ 販路開拓の手段としてウェブサイトを位置づける
```

### 支払い方式

**原則：後払い（精算払のみ）**:
```
Step 1: 自己資金で全額支払い（先出し）
  ↓
Step 2: 事業完了後、実績報告書提出
  ↓
Step 3: 確定検査（書類審査）
  ↓
Step 4: 補助金振込（約2ヶ月後）

⏰ 申請から入金まで：1年以上
```

**注意**:
- ❌ **概算払い制度なし**（ものづくり補助金と異なる）
- ⚠️ 入金完了の通知なし（銀行口座を自分で確認）

### メリット・デメリット

**メリット**:
- ✅ **下限額なし**（100万円の壁がない）
- ✅ **補助率が高い**（2/3、最大3/4）
- ✅ **申請がやや簡単**（商工会議所のサポートあり）
- ✅ **マーケティング費用に重点**
- ✅ **販路開拓に特化**

**デメリット**:
- ❌ **ウェブサイト関連費が1/4まで**（システム開発に不向き）

### 申請プロセス

3. **電子申請**（jGrantsシステム）
4. **採択発表**（申請から約2ヶ月）
5. **事業実施**（交付決定から10ヶ月）
6. **実績報告**（事業完了後30日以内）
7. **補助金入金**（報告から約2ヶ月）

## TaskBridgeへの適用可能性

### ものづくり補助金グローバル枠

**適合度**: ⭐⭐⭐⭐☆（4/5）

**適合する点**:
- ✅ 顧客の100%が海外（北米・欧州企業）
- ✅ 売上計画が明確（12ヶ月で月$8,000の取引目標）
- ✅ 補助額を上回る売上累計の達成見込み
- ✅ 海外市場調査は既存資料（BUSINESS.md、STRATEGY.md）をベースに作成可能
- ✅ 外部専門家（マーケティング、法務）と連携可能

**要確認の点**:
- ⚠️ **プラットフォーム/SaaSの適格性**（最重要）
- ⚠️ 個人事業主でも応募可能か？（法人設立が必要か？）
- ⚠️ 資金調達（200万円以上）の可能性

**向いているケース**:
- 自己資金または融資で200万円以上を調達できる
- プラットフォーム開発費が「システム構築費」として認められる（要確認）
- 大規模なシステム開発が必要

### 小規模事業者持続化補助金

**適合度**: ⭐⭐⭐⭐⭐（5/5）**推奨**

**適合する点**:
- ✅ **下限額なし**で始めやすい
- ✅ **マーケティング重視**（TaskBridgeの本質に合致）
- ✅ 「海外企業向けの販路開拓プラットフォーム」として説明可能
- ✅ 商工会議所のサポートあり
- ✅ 申請難易度が低い
- ✅ 補助率が高い（2/3）

**制約事項**:
- ⚠️ システム開発費は50万円まで
- ⚠️ 単独申請不可（他経費との組み合わせ必須）
- ⚠️ 補助額が小さい（基本50万円）

**向いているケース**:
- マーケティング・販路開拓を中心に進めたい
- システムは自己開発で進める
- 資金調達が限られている（<200万円）
- 初期段階での補助金活用

## 推奨戦略

### 🥇 Option 1: 小規模事業者持続化補助金（最推奨）

**適している理由**:
- ✅ **下限額なし**（最初のハードルが低い）
- ✅ **マーケティング重視**（販路開拓が本質）
- ✅ **申請しやすい**（商工会議所のサポートあり）
- ✅ **補助率が高い**（2/3）
- ✅ **プラットフォーム適格性の確認不要**

**活用戦略**:
```
Phase 1（Month 1-3）: MVP開発
- 自己資金でMVP開発（月$50-65）
- 補助金申請準備（経営計画書作成）
- 商工会議所との連携開始

Phase 2（Month 4-6）: 補助金申請・採択
- 第17回公募に申請（5月1日～6月13日）
- 採択発表（約2ヶ月後）

Phase 3（Month 7-12）: マーケティング強化
- 補助金採択後、事業実施
```

## 付録：FAQ

### Q1: 両方の補助金を同時に申請できますか？

**A**: 同一の経費については、複数の補助金を重複して受けることはできません。ただし、異なる経費項目であれば、理論的には両方に申請可能です。ただし、実務上は商工会議所や認定支援機関に相談することを強く推奨します。

### Q2: 個人事業主でも申請できますか？

**A**:
- **小規模事業者持続化補助金**：✅ 個人事業主も対象
- **ものづくり補助金**：⚠️ 要確認（中小企業庁に確認が必要）

### Q3: 補助金が不採択だった場合、再申請できますか？

**A**: ✅ はい、再申請可能です。不採択理由を確認し、改善して次回の公募に申請できます。

### Q4: 補助金の入金前に資金が不足した場合はどうすればいいですか？

**A**:
- 日本政策金融公庫の融資を検討
- 補助金採択通知を担保に融資を受けられる場合あり
- 商工会議所や金融機関に相談

### Q5: 事業内容を変更したい場合はどうすればいいですか？

**A**: 採択後の大幅な変更は原則不可。軽微な変更は事務局への報告・承認が必要。計画を慎重に作成することが重要。
